# Limits for the Muon Counting SUEP analysis

## Initial setup

To setup the area, do the following:
```bash
wget https://raw.githubusercontent.com/chrispap95/SUEPLimits/refs/heads/main/setup.sh
./setup.sh
cd CMSSW_14_1_0_pre4/src
cmsenv
```

All you need is in the `run_*.sh` scripts (commands to run, examples, etc.).
